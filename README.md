E-Commerce Web Store

Uma aplicação web completa simulando uma loja online, desenvolvida com foco na experiência do utilizador e design responsivo. O projeto abrange todo o fluxo de compras, desde a navegação de produtos até ao checkout e gestão de conta.

Funcionalidades Principais

* **Catálogo de Produtos:** Navegação intuitiva por grelha de produtos com página de detalhes (`produto-detalhe.html`).
* **Gestão de Carrinho:** Adição e remoção de artigos em tempo real (`carrinho.html`).
* **Fluxo de Utilizador:** Sistema completo de Login e Registo de conta.
* **Checkout:** Simulação de processo de pagamento e encomendas.
* **Painel Administrativo:** Dashboard visual para gestão da loja (`admin.html`).
* **Design Responsivo:** Adaptável a diferentes tamanhos de ecrã (CSS personalizado).

Tecnologias Utilizadas

* **HTML5:** Estrutura semântica.
* **CSS3:** Estilização e layout (Flexbox/Grid).
* **JavaScript:** Lógica de interação no frontend.

## Como Usar

1.  Faça o download ou clone este repositório.
2.  Abra o ficheiro `index.html` no seu navegador (Chrome, Edge, Firefox).
3.  Navegue pela loja para testar as funcionalidades.
